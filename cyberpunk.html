Role

  • Motion Designer

Project Type

Tools Used

  • Adobe After Effects
  • Adobe Illustrator
  • Adobe Photoshop

Team

  • Nathanael-Motion Designer

Duration

  • Nov 2023 - Jan 2024

I was a really big fan of the anime Cyberpunk 2077 Edge Runners , and I wanted to brush up on my motion design skills. So I set out to recreate the opening credits for the show for one of my typography assignments. I started this project off by seeing what effects and assets I would need to create this intro, and seeing what I already knew and what I would have to learn. After I gathered what new things I would have to learn to accomplish this I started working away!

When I started to work on this project I started to create visual elements that I would need and gather media that would also be a part of the project. This involved using the same color scheme, correct fonts and using effects that match those shown in the original intro.

Overall I am very proud of how this project turned out, you can watch the full video here. The product that I have made almost entirely is the exact same as the media I based it on.
